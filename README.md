# CLUSTERINNODE
## Installation

```bash
$ npm init
$ npm i colors
```
## Running the app

```bash
# run
$ node cluster.js

# SIGUSR2 exit code
$ kill -s SIGUSR2 *Your Procces Pid*

# SIGINT exit code
$ kill *Your Procces Pid*
```
